== Description ==
Flormar Slider is a simple WooCommerce product slider plugin that displays the best-selling products in a beautiful carousel format.

== Installation ==
1. Download the plugin ZIP file.
2. Upload it to the `/wp-content/plugins/` directory via FTP, or install it through the WordPress Plugin Manager.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Use the shortcode echo do_shortcode('[flormar-test-slider]'); on any page or post to display the slider.

== Shortcode Usage ==
`[flormar-test-slider max-price="100" min-price="20"]`

- `max-price` (optional): Set a maximum price filter for products.
- `min-price` (optional): Set a minimum price filter for products.

== Features ==
- Displays WooCommerce products in a slider.
- Supports filtering by minimum and maximum price.
- Uses Slick Carousel for smooth sliding effects.
- Fully responsive for all screen sizes.
